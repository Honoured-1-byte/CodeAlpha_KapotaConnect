# KapotaConnect – Mini Social Media App

A lightweight social platform where users can:
- Sign up / log in
- Create posts & comment
- Like posts
- Follow / unfollow other users

## Getting Started
1. `git clone <your‑new‑repo‑URL>`
2. `npm install`
3. Fill in `.env` (DB URL, session secret, etc)
4. `npm start`
